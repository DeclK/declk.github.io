SA-SSD note

这篇文章的重点之一就是引入了 auxiliary network 来帮助学习，在之后的SE-SSD中也受其启发

Introduction

We design an auxiliary network which converts the convolutional features in the backbone network back to point-level representations.

The auxiliary network can be detached after training and therefore introduces no extra computation in the inference stage.

Specifically, the auxiliary network first converts the features from the backbone network back to point-wise representations and then performs two auxiliary tasks:

  1. foreground segmentation to drift the features to be sensitive to the object boundary and
  2. point-wise center estimation to make the features be aware of the intra-object relationship.

The auxiliary network is jointly optimized with the backbone network in training stage and is removed after training, introducing no extra computational cost at inference stage.

Contributions

(1) We propose a structure-aware single-stage 3D object detector, which employs a detachable auxiliary network

(2) We develop an efficient feature map warping method to mitigate the discordance between the predicted bounding boxes and corresponding classification confidences

Part-sensitive warping

起因：在单阶段检测中， feature map 和 anchor 的对齐问题是普遍存在的问题， 这样会导致预测出来的边界框的定位质量与置信度不匹配，这会影响在后处理阶段（NMS）时， 高置信度但低定位质量的框被保留， 而定位质量高却置信度低的框被丢弃。在 two-stage 的目标检测算法中，RPN 提取 proposal，然后会在 feature map 上对应的的位置提取特征（roi-pooling 或者 roi-align），这个时候新的特征和对应的 proposal 是对齐的

解决思想：让 bounding box 的 confidence prediction 来自于 bounding box 位置对应的 classification map，而不是来自于 anchor，具体步骤如下

  1. 利用卷积网络将 classification map 升维至 K 维，K 是bounding box分割的份数，之后会详细提到

    modify the last classification layers to generate K part-sensitive classification maps, denoted by ${X_k : k = 1, 2, …, K}$, each of which encodes information of a certain part of the object

  2. 将预测的 bounding box (xy平面投影) 均匀分成 K 个部分（文章分成了 4x7=28 个部分），取每个部分的中心点作为采样点 ${S_k : k = 1, 2, …, K}$ ，将 K 个采样点的 (classfication) confidence 的平均值作为最终结果。由于做回归的 feature map 和 classification map 是具有对应关系的，我们直接找到采样点在 classification map 中对应的点，通过 bilinear interpolation 就能得到该点的 confidence。需要注意的是因为我们将 classification map 升维到 K 维，每个采样点的 confidence 对应着 K 维中的一个。
